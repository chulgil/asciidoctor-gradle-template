== Asciidoc テンプレート集

=== 脚注

脚注は次のように書きます。

[WARNING]
====
``WARNING`` メッセージはここに。
====

[TIP]
====
``TIP`` メッセージはここに。
====

その他に、``IMPORTANT`` ``NOTE`` ``CAUTION`` があります。

1行でこのようにもかけます。

[source]
IMPORTANT: 重要事項を記述します。

=== キーボードショートカット表記

|===
|ショートカット|目的
|kbd:[F11]
|全画面表示
|kbd:[Ctrl+T]
|新規タブを開く
|kbd:[Ctrl+Shift+N]
|シークレットウインドウを開く
|kbd:[Ctrl + +]
|ズーム

ファイルを保存するには menu:File[Save] を選択します。拡大率をデフォルトに戻すには menu:View[Zoom > Reset] を選択します。

完了したら btn:[OK] ボタンを押してください。ナビゲーターでファイルを選択し、 btn:[開く] をクリックしてください。

=== ソースコード（直接記述）

[source, java]
.Hello.java
----
class Hello {
    public static void main(String[] args) {
        // comment
        System.out.println("Hello, World"); // <1>
    }
}
----

<1> 注釈をいれることができます。

=== ソースコード（外部ファイルのinclude）

NOTE: ファイルから行数とインデントを指定できます。

[source, javascript, indent=0]
.sample.js
----
include::src/sample.js[lines=2..3]
----

<1> include 中でも注釈を指定できます。
